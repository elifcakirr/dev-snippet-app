# 🚀 Dev Snippet App

Kod parçacıklarını (snippet) kaydetmek, düzenlemek ve yönetmek için geliştirilmiş basit ve kullanışlı bir uygulama.

## ✨ Özellikler

* 📌 Tekrar kullanılabilir kod parçacıkları ekleme
* 🔍 Snippet arama ve filtreleme
* 🗂️ Kategori veya etiketlere göre düzenleme
* 💻 Geliştirici dostu sade arayüz
* ⚡ Hızlı ve hafif kullanım

## 🛠️ Kullanılan Teknolojiler

* Frontend: (React / HTML-CSS-JS)
* Backend: (Next.js)
* Veritabanı: (Mysql)

## 📦 Kurulum

Projeyi klonla:

```bash
git clone https://github.com/elifcakirr/dev-snippet-app.git
```

Proje klasörüne gir:

```bash
cd dev-snippet-app
```

Bağımlılıkları yükle:

```bash
npm install
```

Uygulamayı başlat:

```bash
npm start
```

## 🚀 Kullanım

* Yeni snippet ekleyebilirsin
* Mevcut snippet’ları düzenleyebilir veya silebilirsin
* Anahtar kelimelerle arama yapabilirsin
* Snippet’larını düzenli şekilde saklayabilirsin
